About

This is a project the course M7012E - Pervasive Computing held at Lule√• University of Technology. The aim of our project is to build a simple wearable camera which will timelaps the wearers day. The pictures will be stored on a server, which can be acced by the user. The camera will also be configurable via a simple app allowing the user to change interval between pictures, turn the camera on and off etc.

To implement all of this, we will be using a Raspberry Pi Zero W equiped with a camera and battery. The app will be an android app and the server will run on a different Raspberry Pi which in turn runs Docker. This can of course be further developed but this is the main idea.
